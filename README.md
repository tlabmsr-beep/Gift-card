# KASHEMIR · Подарочные карты

Сайт оформления и оплаты подарочных карт студии красоты **KASHEMIR** (Нижний Новгород).
Поток из 6 шагов в стиле бренда, оплата через **Тинькофф Касса**, демо-режим из коробки.

## Что внутри

```
kashemir-giftcard/
├── server/
│   ├── server.js        Express: создание платежа, webhook, статусы
│   ├── tinkoff.js       Интеграция с Тинькофф (Init / Token / GetState / проверка webhook)
│   └── .env.example     Шаблон конфигурации
├── public/
│   ├── index.html       6 шагов оформления
│   ├── styles.css       Фирменный стиль KASHEMIR
│   ├── app.js           Логика шагов, карусель, валидация, оплата
│   ├── cards.js         6 SVG-дизайнов карт из элементов бренда
│   ├── demo-pay.html    Тестовая платёжная страница
│   └── success.html     Страница «карта оплачена»
└── package.json
```

## Запуск (демо-режим)

```bash
npm install
npm start
```

Откройте **http://localhost:3000**. Без ключей сайт работает в тестовом режиме:
кнопка «оплатить» ведёт на демо-страницу оплаты, ничего не списывается — можно
пройти весь поток целиком.

## Подключение реальной оплаты (Тинькофф Касса)

1. Зарегистрируйтесь в **Тинькофф Касса** (нужны ИП или юрлицо).
2. В личном кабинете → «Магазины» → «Терминалы» возьмите **TerminalKey** и **SecretKey**.
3. Скопируйте `server/.env.example` в `server/.env` и заполните:

   ```env
   TERMINAL_KEY=ваш_terminal_key
   SECRET_KEY=ваш_secret_key
   DEMO_MODE=false
   BASE_URL=https://ваш-домен.ru
   ```

4. В кабинете Тинькофф укажите адрес уведомлений (webhook):
   `https://ваш-домен.ru/api/tinkoff-webhook`
5. Перезапустите сервер. Теперь «оплатить» открывает настоящую платёжную форму Тинькофф.

При оплате Тинькофф вызывает `/api/tinkoff-webhook`. Когда приходит статус
`CONFIRMED`, заказ помечается оплаченным (см. `server.js` — там отмечено
местом `TODO`, куда добавить отправку карты по СМС или PDF на почту).

## Что стоит доделать перед продакшеном

- **База данных** вместо `orders` в памяти (заказы сейчас теряются при перезапуске).
- **Выпуск карты**: отправка СМС со ссылкой и/или генерация PDF на e-mail после `CONFIRMED`.
- **Чек 54-ФЗ**: блок `Receipt` в `tinkoff.js` заполнен базово — сверьте налог
  (`Taxation`, `Tax`) со своей системой налогообложения.
- **HTTPS** обязателен для боевого webhook и платёжной формы.

## Фирменный стиль

Цвета: лесной зелёный `#3D7337`, тёмная хвоя `#21432A`, золотой песок `#DABD91`,
кремовый `#FFF2D0`, графит `#231E16`. Шрифты: Cinzel (заголовки) + Manrope (текст).
Дизайны карт собраны из мотивов бренда: кольцо-медальон, листья, соты, золотые линии.
